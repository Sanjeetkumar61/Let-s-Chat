# 💬 Let's Chat - Real-Time Chat Application

A full-stack real-time chat application built using the **MERN Stack** and **Socket.IO**. The application provides seamless one-to-one and group messaging with real-time updates, online user status, typing indicators, file sharing, and authentication.

## 🚀 Live Demo

🌐 **Application:** https://let-s-chat-iota.vercel.app/

---

## 📸 Features

- 🔐 JWT Authentication (Register & Login)
- 💬 One-to-One Real-Time Chat
- 👥 Group Chat
- ⚡ Real-Time Messaging using Socket.IO
- 🟢 Online / Offline User Status
- ✍️ Typing Indicator
- 📂 File & Image Sharing (Cloudinary)
- 📥 File Download & Preview
- ✅ Message Delivery Status
- 👀 Read Receipts
- 🗑 Delete Message
  - Delete for Me
  - Delete for Everyone
- 🔍 Search Messages
- 📱 Responsive UI
- 🎨 Modern User Interface
